# Retail Business Performance Insights (2023-2026)

This project focuses on analyzing retail sales data to derive actionable business insights. Using Tableau, I have developed an interactive dashboard to visualize sales trends and category-wise profitability.

## 📊 Project Objective
The primary objective of this dashboard is to analyze three years of sales performance. This helps in identifying high-growth areas and profit centers, supporting data-driven decision-making for retail operations.

## 🛠 Tools Used
* **Tableau:** For data visualization and dashboard design.
* **Analysis:** Trend analysis, category-wise performance metrics, and profitability mapping.

## 📈 Key Insights
* **Sales Growth:** Observed a consistent upward trend in sales performance from 2023 to 2026.
* **Top Performing Category:** 'Technology' emerged as the most profitable category.
* **Design Approach:** The dashboard follows cognitive and visual design principles to ensure clarity and ease of use for non-technical stakeholders.
